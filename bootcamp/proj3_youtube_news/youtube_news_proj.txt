유튜브에서 특정 뉴스채널(YTN, SBS, MBC 등..)의 음성 파일을 불러와서 텍스트로 변환 후 "요약"하여 사용자에게 제공

최신순으로 5개씩 2분이하의 동영상을...?

그 중 내용(키워드)이 "유사한 정보"를 선정하여 사용자에게 제공..?


취지 : 당장 사용 가능한 서비스를 만들기 보다는 부트캠프에서 배운 내용들을 최대한 활용해 보자에 초점을 맞춰서 이번 프로젝트를 진행함


사용 기술...?
1. youtube api를 이용해 최신 영상의 mp3파일 불러오기(채널ID, 영상ID 불러오기)
2. whisper 이용하여 음성을 텍스트로 변환
3-1. "keybert 모델"을 사용하여 텍스트에서 키워드 추출
 - 첫번째는 직접 모델을 불러와서 파이썬에서 바로 적용 -> 키워드 바로 제공
 - 두번째는 onnx파일로 모델을 가져와서 "openvino"로 최적화 -> 임베딩 벡터 형태로 제공
3-2. 텍스트를 토큰화하여 추출된 임베딩 벡터로 유사도 계산하여 키워드 추출
4. openai api 이용해서 chat gpt 4를 이용해서 키워드 기반으로 문장 생성
5. 최신 기술인 llama3.2도 사용해 보기 (hugging face에서 모델 가져오고 오픈비노로 최적화 했는데 너무 느림)

BERT-based 모델 (KeyBERT 등):
BERT와 같은 언어 모델을 기반으로 한 키워드 추출. 문맥을 이해하면서 의미 있는 키워드를 뽑아낼 수 있어. 특히 문맥적 의미가 중요한 뉴스 콘텐츠나 복잡한 문서에서 좋은 성능을 발휘해.
GPT-기반 추출 방법:
GPT 계열 모델을 활용하여 특정 주제에 맞는 키워드를 추출하는 방식. 이 경우 미리 훈련된 모델을 활용해 특정한 패턴에 맞는 키워드를 뽑아낼 수 있어.
